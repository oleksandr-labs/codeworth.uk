## Концепція та дизайн

**Стиль:** Ультра-мінімалістичний чорно-білий — чорний (`#000000`), білий (`#FFFFFF`),
один акцентний колір: теплий amber (`#F59E0B`) лише для CTA та активних елементів.
Весь акцент — на фотографіях. UI буквально зникає. Натхнення: Peter McKinnon, Jose Villa.

**Ключова відмінність:**
- Найбільш мінімалістичний UI у портфоліо — фото IS the design
- Masonry grid для портфоліо (Pinterest-стиль)
- Fullscreen lightbox з клавіатурною навігацією
- "Book a Session" flow з вибором типу зйомки та дати
