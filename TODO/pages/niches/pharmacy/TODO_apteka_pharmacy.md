# Apteka.ua — Онлайн-аптека з доставкою

**Портфоліо-проєкт:** `portfolio/online-pharmacy`
**Ніша:** Аптека / Pharmacy
**Клієнт:** Мережа аптек з онлайн-замовленням, Харків
**Складність:** 🔴 Складна
**Стек:** Next.js 15, TypeScript, Tailwind CSS v4, Zustand (cart)
**Статус:** ✅ Реалізовано — реалізовано у демо-компоненті `FarmaPlusDemo.tsx`

## Концепція та дизайн

**Стиль:** Медично-чистий — зелений (`#16A34A`), білий (`#FFFFFF`), світло-сірий (`#F8FAFC`).
Строгий та зрозумілий — аптека викликає довіру через порядок та читабельність.
Натхнення: Liki24, Tabletki.ua. Жодного агресивного маркетингу — акцент на здоров'я.

**Ключова відмінність:**
- Аналог маркетплейсу ліків з каталогом, кошиком, пошуком
- Завантаження рецепту (upload + review)
- "Аналоги" — показ дешевших замінників
- Перевірка взаємодії ліків (drug interaction checker — simplified UI)
