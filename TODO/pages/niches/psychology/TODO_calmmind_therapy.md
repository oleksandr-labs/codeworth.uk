Натхнення: академічні психологічні платформи, Psychology Today. Відчуття: "тут є наукове підґрунтя, не просто розмови".

## Key Sections & Features

### Що таке КПТ (Education)

- [x] Доступне пояснення: «КПТ = думки → емоції → поведінка»
- [x] Схема-ілюстрація когнітивної моделі (SVG)
- [x] «З чим допомагає КПТ»: тривожні розлади / ОКР / фобії / депресія / ПТСР
- [x] «Скільки сесій потрібно» — чесна відповідь (8–20 залежно від запиту)

### Self-Assessment тести

- [x] GAD-7 (тривога): 7 запитань, 0–4 балів, автоматична інтерпретація
- [x] PHQ-9 (депресія): 9 запитань, з рекомендацією звернутись до спеціаліста
- [x] Результат: «Ваш рівень тривоги — помірний. Ось що це означає...»
- [x] Дисклеймер: «Тест не є діагнозом. Зверніться до спеціаліста.»
- [x] CTA після тесту: «Записатись на безкоштовну 20-хв консультацію»

### Каталог терапевтів

- [x] Картки: фото + ім'я + ступінь + спеціалізація + мова + ціна сесії
- [x] Фільтр: Запит / Мова / Формат (онлайн/офлайн) / Вік клієнта
- [x] Профіль: bio + підхід + освіта + сертифікати + відгуки + розклад

### Групова терапія

- [x] Активні групи: тема + день + час + кількість місць + ведучий
- [x] Теми: «Подолання тривоги» / «Управління гнівом» / «Самооцінка»
- [x] «Перше заняття — безкоштовно» для нових учасників
