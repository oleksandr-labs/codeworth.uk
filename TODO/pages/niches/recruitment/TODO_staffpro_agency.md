## Key Sections & Features

### Job Board (Вакансії)

- [x] Пошук: посада + місто + галузь
- [x] Фільтри: Зайнятість (повна / часткова / тимчасова) / Галузь / Зарплата / Досвід
- [x] Картки: посада + компанія (анонімно або відкрито) + місто + зарплата + дата + «Відгукнутись»
- [x] «Швидкий відгук»: заповнити форму без CV (ім'я + телефон)
- [x] «Нові вакансії сьогодні» окремий розділ

### Для Роботодавців

- [x] «Чому ми»: швидкість (72 год) + ціна (фікс або % від оклада) + гарантія
- [x] Галузі: HoReCa / Ритейл / Логістика та склади / Виробництво / Адміністрація / IT
- [x] Тарифи: Стандарт (1 вакансія) / Пакет (5 вакансій) / Масовий (20+)
- [x] Форма: компанія + посада + к-сть + терміни + умови

### Масовий Найм

- [x] «Нам потрібно 50 касирів» — ми закриємо за 2 тижні
