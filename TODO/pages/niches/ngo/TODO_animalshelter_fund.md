теплий білий (`#FFFBF7`). Великі фото вихованців (котів, собак), округлі форми, емоційний дизайн.
Натхнення: Humane Society, HSUS. Відчуття: "кожна тварина заслуговує на дім — допоможи їй знайти".

## Key Sections & Features

### Каталог тварин для усиновлення

- [x] Великі картки: фото + ім'я + вид + вік + «Чекає вже X днів» + «Познайомитись»
- [x] Фільтр: Кіт / Собака / Вік / Розмір / Характер (активний / спокійний / для дітей)
- [x] «Познайомити» функція — не «усиновити»: виїзд або візит до притулку
- [x] Кожен профіль: 5+ фото + story + характер + вакцинований/стерилізований/чипований

### Емоційні профілі тварин

- [x] «Мене звуть Сонечко. Мені 2 роки. Мене знайшли в підвалі...»
- [x] «Я шукаю людину, яка...» — короткий текст від «імені» тварини
- [x] «Поділитись» → соцмережі (viral sharing)
- [x] «Стати фостером для Сонечки» кнопка

### Донат

- [x] Суми: 50 / 100 / 300 / 500 ₴ + своя
- [x] Разовий / Щомісячний
- [x] Призначення: Корм / Лікування / Стерилізація / Де потрібніше
- [x] «Що дає X гривень»: «200 ₴ = 1 місяць корму для одного кота»
- [ ] Реальний LiqPay / Fondy

### Фостерна сім'я

- [x] «Тимчасовий дім = шанс на усиновлення»
