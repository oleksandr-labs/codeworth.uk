теплий жовтий (`#FDE68A`), білий фон. Акварельні плями та розмиття як декоративні елементи.
Натхнення: арт-студії для дітей Японії та Барселони. Відчуття: "тут можна бруднити руки та створювати шедеври".

**Ключова відмінність:**

- Галерея дитячих робіт — батьки можуть «замовити» роботу дитини у рамці
- Майстер-класи у форматі «Тематичний день»
- Творчі вечори для дорослих (батьки + діти)

## Key Sections & Features

### Гуртки

- [x] Картки: ілюстрація напрямку + назва + вік + к-сть занять/міс + ціна
- [x] Напрямки: Акварель / Масло / Ліплення з глини / Графіка / Змішана техніка / Батик
- [x] Підхід: «Не ставимо оцінки — підтримуємо інтерес»
- [x] Розмір групи: «Не більше 8 дітей»

### Майстер-класи

- [x] Найближчі МК: тема + дата + вік + к-сть місць + ціна
- [x] Теми: «Морський світ» / «Казковий ліс» / «Мій герой» / «Космос»
- [x] «Подарунковий сертифікат на МК» — популярний подарунок
