# FamilyGuard — Сімейний адвокат та розлучення

**Портфоліо-проєкт:** `portfolio/family-law`
**Ніша:** Юридичні послуги / Family Law
**Клієнт:** Сімейний адвокат, Одеса
**Складність:** 🟡 Середня
**Стек:** Next.js 15, TypeScript, Tailwind CSS v4
**Статус:** ✅ Реалізовано

---

## Концепція та дизайн

**Стиль:** Теплий та емпатичний — кремово-бежевий (`#FDF6EC`), теракотовий акцент (`#C4704A`),
темний шоколад (`#2D1B0E`) для тексту, ніжно-зелений (`#D1E8D5`) для підтримуючих елементів.
M'які rounded corners, sans-serif (гуманістичний, не холодний), живі фото — руки, люди, не суди.
Натхнення: BetterHelp, Knotel. Відчуття: "тут вас почують і допоможуть, без зайвої формальності".

**Ключова відмінність:**
- Емпатичний тон усього UX — ніяких холодних юридичних термінів у першому екрані
- Anonymous Consultation: клієнт описує ситуацію без імені до першого контакту
- "Ваші права" калькулятор: стаж шлюбу → спрощена інформація про поділ майна
- Повна протилежність корпоративному LexUA — людяність замість авторитету

## Anonymous Consultation Flow

- [x] "Анонімна консультація" — без прізвища та телефону на першому кроці
- [x] Крок 1: Опис ситуації (textarea, до 500 символів) — тільки суть
- [x] Крок 2: Тип питання (dropdown: розлучення / майно / діти / аліменти / інше)
- [x] Крок 3: Контакт для відповіді (email або Telegram — клієнт обирає)
- [x] Підтвердження: "Адвокат ознайомиться із ситуацією та напише вам протягом 24 год"
