- [x] Впровадження CRM: KeyCRM, HubSpot, Zoho, кастомна — реалізовано через `service.longDescription` + `service.features` ("Кастомна під бізнес") у `services.ts`
- [x] Автоматизація воронки продажів — реалізовано через `service.includes` ("Автоматизація воронки продажів") у `services.ts`
- [x] Інтеграція CRM з сайтом та месенджерами — реалізовано через `service.features` ("Інтеграції: Telegram, Нова Пошта, LiqPay, Google Sheets, Zapier") у `services.ts`
- [x] Кастомна ERP-система для специфічного бізнесу — реалізовано через `service.packages` ("ERP / Platform") у `services.ts`
- [x] Кабінет з особистими кабінетами клієнтів — ✅ додано до `service.useCases` ("Кабінет клієнта") у `services.ts`
