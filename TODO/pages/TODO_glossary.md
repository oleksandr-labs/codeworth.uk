### Структура `glossary.ts`
```ts
type GlossaryTerm = {
  slug: string;         // "core-web-vitals"
  termUk: string;       // "Core Web Vitals"
  termEn: string;       // "Core Web Vitals"
  category: GlossaryCategory;
  shortDescription: string;  // 1-2 речення для списку
```

### Примітки
- Глосарій — потужний інструмент для **topical authority**: Google сприймає сайт як експерта в ніші.
- Кожен термін може отримувати трафік за запитом "що таке X" — таких запитів тисячі на місяць.

- Можна автоматично ставити посилання-тултіп на перше вживання терміну в блозі.
