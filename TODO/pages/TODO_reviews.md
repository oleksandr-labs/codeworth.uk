```ts
type Review = {
  id: string;
  authorName: string;
  authorRole: string;        // "Власник ресторану «Зелений Лист»"
  authorPhoto?: string;      // шлях до фото
  company?: string;
  companyUrl?: string;
  rating: 1 | 2 | 3 | 4 | 5;
  text: string;              // основний текст відгуку
  resultHighlight?: string;  // "Трафік +180% за 3 місяці" — виділений результат
  serviceSlug?: string;      // яка послуга надавалась
  nicheSlug?: string;        // яка ніша
  date: string;              // "2025-11-15"
  platform?: "google" | "clutch" | "dou" | "direct"; // джерело
  verified: boolean;
};
```

### Примітки
- Сторінка відгуків — критичний E-E-A-T сигнал: Google оцінює "реальний досвід" як фактор ранжирування.
- Rich Snippet зірок у видачі підвищує CTR на 20–35%.
- Регулярно просити клієнтів залишити відгук після закриття проєкту (автоматизувати через CRM).
- Навіть 10–15 якісних відгуків на старті краще, ніж порожня сторінка.
