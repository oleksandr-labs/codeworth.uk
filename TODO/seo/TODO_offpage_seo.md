 Позасайтове SEO (Off-Page SEO)
Опис: Нарощування авторитету домену через зворотні посилання, PR, цитування та згадки бренду.
**Статус:** Не розпочато

---

> ⚠️ **Незакреслені пункти ([ ]) — Phase 2:** потребують реального трафіку, акаунтів у SEO-інструментах або реєстрації в зовнішніх сервісах. Технічний SEO (sitemap, hreflang, schema) вже реалізовано у коді.


## Аналіз поточного профілю посилань
- [ ] Перевірити Domain Rating (DR) через Ahrefs або Moz після деплою
- [ ] Перевірити кількість referring domains та backlinks
- [ ] Виявити токсичні посилання через Ahrefs / Google Search Console → Disavow

## Link Building — EN ринок (Великобританія, пріоритет)
- [ ] Реєстрація в авторитетних UK бізнес-каталогах: Yell.com, Thomson Local, Scoot, FreeIndex
- [ ] Clutch.co — профіль агенції з кейсами та відгуками (EN)
- [ ] DesignRush — лістинг агенції
- [ ] GoodFirms — профіль з портфоліо та відгуками
- [ ] Bark.com — профіль підрядника (UK market)
- [ ] DMOZ-альтернативи: Best Web Design Agencies UK directories
- [ ] ProductHunt — запуск маркетплейсу / продуктів CodeNest

## Link Building — UK ринок (Україна, вторинний)

## Guest Posting (Гостьові публікації)
- [ ] Скласти список EN блогів для гостьових статей (web design, SaaS, SMB, UK startup blogs)
- [ ] Скласти список UA блогів / медіа для гостьових статей (DOU, AIN, MC.today)
- [ ] Написати 2-3 статті на тиждень для публікації на зовнішніх майданчиках
- [ ] Шаблон pitch-листа для редакторів (EN + UK версії)
- [ ] Відстежувати published guest posts + отримані посилання в таблиці

## Моніторинг згадок бренду

### Примітки
- EN ринок (UK) — пріоритет. 1 якісний UK backlink > 10 загальних.
- Уникати PBN, платних посилань, link farms — Google Penalty.
- Мета: DR 30+ за перший рік. Переглядати профіль посилань щомісяця.
