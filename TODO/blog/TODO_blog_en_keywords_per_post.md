## Як використовувати цей файл

1. Перед написанням поста — взяти EN title, EN keyword, meta description
2. EN Title вставити в `title.en` поля `BlogPost`
3. EN keyword = перші слова в `description.en`
4. Meta description = `description.en` (145-155 chars)
5. Після публікації: відмітити ✅
