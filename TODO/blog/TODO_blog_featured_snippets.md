# ⭐ Featured Snippets & People Also Ask Strategy

> 50 цільових Featured Snippets та PAA питань для блогу CodeNest.
> Ціль: Position 0 + PAA блоки → безкоштовний топ-трафік без backlinks.

---

## Що таке Featured Snippet та чому це важливо

- **Featured Snippet** = відповідь показується над органічною видачею (Position 0)
- **People Also Ask (PAA)** = розкривні питання в SERP — кожна відповідь генерує нові PAA
- **AI Overview** = Google SGE використовує structured content — FAQ schema критична
- CTR для Featured Snippet: **8-12%** vs 3-5% для звичайної позиції 1
- Для отримання: **пряма відповідь у перших 40-50 словах після H2**

---

## Формати Featured Snippets (як писати контент)

### 1. Paragraph Snippet (What is X?)
```
## What is [Term]?
[Term] is [definition in 40-50 words]. It [explains key benefit/use case].
For example, [concrete example]. [One more sentence of context].
```

### 2. List Snippet (Steps / How to)
```
## How to [Do X]: Step-by-Step
1. [Step 1 — verb + noun, max 10 words]
2. [Step 2]
3. [Step 3]
...
```

### 3. Table Snippet (Comparisons)
```
| Feature | Option A | Option B |
|---------|---------|---------|
| Price | £X | £Y |
| Speed | Fast | Slow |
```

### 4. FAQ Schema (People Also Ask targets)
```tsx
// Кожен пост, що має FAQ-секцію:
// Мінімум 5 питань з прямими відповідями ≤ 50 слів кожна
// Schema: FAQPage з Question + acceptedAnswer
```

## People Also Ask (PAA) Expansion Strategy

### Як отримати PAA-блок
1. Знайти PAA питання для target keyword (GSC → Queries з "?" )
2. Написати **точну відповідь у 2-3 реченнях** після відповідного H2
3. Додати це питання до FAQ-секції поста (FAQPage schema)
4. Кожна PAA-відповідь генерує нові PAA → snowball ефект

## Schema.org FAQPage Implementation

### Для кожного поста з FAQ-секцією
```tsx
// src/app/[lang]/blog/[slug]/page.tsx
const faqSchema = {
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": post.faq?.map(item => ({
    "@type": "Question",
    "name": item.question,
    "acceptedAnswer": {
      "@type": "Answer",
      "text": item.answer  // max 200 chars для snippet
    }
  }))
}
```

### Нова структура `BlogPost.faq`
```typescript
interface BlogPost {
  // ...existing fields
  faq?: Array<{
    question: { en: string; uk: string }
    answer: { en: string; uk: string }
  }>
}
```
